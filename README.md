# Implementation of Conway's Game of Life

### Steps for Setting up the App:
* Clone the Repository to your Local Machine
* Run the Following Command:
 ```bash
 npm install 
 ```
 * Once its finished simply do:
 
  ```bash
 npm start 
 ```
